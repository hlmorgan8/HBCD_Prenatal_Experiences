# Protective Prenatal Experiences Project 

## Getting Started

To reproduce results from DCN Paper:
```
Rscript load_data.R 
Rscript build_variables.R 
Rscript demographics.R
Rscript linear_model_PACES.R
Rscript linear_model_Depression.R
```
This assumes you have the HBCD dataset available to you in `\data` folder
