# Prenatal Experiences Project 

## Getting Started

To reproduce results:
```
Rscript load_data.R 
Rscript build_variables.R 
Rscript demographics.R
Rscript correlation_mriQC.R
Rscript linear_model_Depression.R
Rscript linear_model_postnatal.R
Rscript interaction_medication.R
```
This assumes you have the HBCD dataset available to you in `\data` folder
